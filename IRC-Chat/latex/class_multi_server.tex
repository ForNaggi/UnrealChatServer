\doxysubsection{상세한 설명}
select 기반 루프를 통해 다중 클라이언트 채팅 서버를 관리하는 클래스입니다. 

\doxylink{class_multi_server}{Multi\+Server} 클래스는 지정된 포트에서 서버를 시작합니다. ~\newline
신규 클라이언트 연결을 받아들입니다. ~\newline
등록된 클라이언트로부터 메시지를 수신합니다. (\doxylink{class_message_receiver}{Message\+Receiver} 사용). ~\newline
등록된 클라이언트들에게 메시지를 발송합니다. (\doxylink{class_message_sender}{Message\+Sender} 사용). ~\newline
서버 동작 전반을 관리합니다. ~\newline
각 단계에서의 오류를 나타내는 결과 상태 값(\+Result)을 제공합니다.
