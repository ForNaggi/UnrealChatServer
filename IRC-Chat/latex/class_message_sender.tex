\doxysubsection{상세한 설명}
서버에서 클라이언트 소켓으로 메시지를 보내는 기능을 제공하는 클래스입니다. 

이 클래스는 서버 메시지의 다양한 전달 모드를 구현합니다.
\begin{DoxyItemize}
\item Broadcast \+: 모든 연결된 클라이언트에게 메시지 전송.
\item Multicast \+: 특정 클라이언트를 제외한 다수에게 메시지 전송.
\item Unicast \+: 한 클라이언트에게만 메세지 전송.
\end{DoxyItemize}

내부적으로 개행 문자(NEW\+\_\+\+LINE 상수)를 메시지 끝에 추가하여 포맷팅합니다. ~\newline
전송 작업 결과를 나타내는 \doxylink{class_message_sender_a39e62bd3398a07e5fddfa8f1e4d4b0f6}{Result} 열거형을 사용합니다.
