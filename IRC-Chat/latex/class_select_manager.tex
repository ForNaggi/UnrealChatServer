\doxysubsection{상세한 설명}
소켓 집합을 관리하고 select()를 사용하여 네트워크 I/\+O를 다중화합니다. 

Select\+Manager는 select에 필요한 fd\+\_\+set 구조체를 래핑합니다. ~\newline
소켓을 추가하여 select 연산을 수행하는 메서드를 제공합니다. ~\newline
어떤 소켓에 수신 데이터가 있는지(읽기 준비 완료 상태인지)를 확인합니다. ~\newline
타임아웃 상황을 처리할 수 있습니다.
