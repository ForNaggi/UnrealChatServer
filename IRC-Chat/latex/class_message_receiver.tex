\doxysubsection{상세한 설명}
클라이언트 소켓으로부터 들어오는 메시지를 처리하는 클래스입니다. 

클라이언트 소켓을 전달받아 객체를 생성합니다. ~\newline
소켓으로부터 메시지를 수신하여 저장하고, 특별한 명령(예\+: 종료 요청)이 있는지 확인합니다. ~\newline
고정 크기 버퍼를 사용하여 수신된 데이터를 처리합니다.
