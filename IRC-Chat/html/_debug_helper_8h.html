◆ LogLevel

enum class LogLevel
strong

로그의 출력 유형을 정의합니다.

각 레벨은 로그 메시지의 중요도나 유형을 나타냅니다.
INFO는 일반, WARNING은 경고, ERROR는 오류, DEBUG는 디버깅용 메시지.

열거형 멤버
INFO 

일반 메세지 레벨.

WARNING 

경고 메시지 레벨.

ERRORZ 

오류 메시지 레벨.

DEBUG 

디버그 메시지 레벨.

◆ log()

void log ( LogLevel level,
const std::string & message,
const char * file = "",
int line = 0 )
inline

주어진 로그 레벨, 메시지를 로그 유형에 따라 파일명, 라인위치 정보와 함께 로그로 출력합니다.

매개변수
[IN]LogLevel level : 로그 레벨 (INFO, WARNING, ERRORZ, DEBUG).
[IN]const std::string& message : 로그로 출력할 메시지 텍스트.
[IN(default: "")] const char* file : 로그 발생 소스 파일 이름 (ERRORZ/DEBUG 레벨에서 사용).
[IN(default: 0)] int line : 로그 발생 소스 코드 줄 번호 (ERRORZ/DEBUG 레벨에서 사용).
반환값
없음.

로그 레벨에 따라 출력 스트림을 결정합니다 (오류는 std::cerr, 그 외는 std::cout). 로그 앞부분에 타임스탬프와 레벨을 붙이고, ERRORZ와 DEBUG 레벨에서는 파일명과 라인 번호를 추가합니다. 디버그 모드가 아닌 빌드에서는 DEBUG 레벨 메시지는 출력되지 않습니다.
